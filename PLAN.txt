2. Usage Extraction (critical piece)
Strategy:
Execute JS inside WebView
Extract % from DOM
Minimal version (works surprisingly well)
export async function getUsagePercent(webview) {
  const result = await webview.eval(`
    (() => {
      const match = document.body.innerText.match(/(\\d+)%/);
      return match ? parseInt(match[1]) : null;
    })()
  `);

  return result;
}
Better version (more stable)

After inspecting the page:

Target a specific label like “Usage”

3. Polling Loop
let lastPercent = null;

setInterval(async () => {
  const percent = await getUsagePercent(webview);

  if (percent == null) return;

  if (percent !== lastPercent) {
    updateTray(percent);
    checkThresholds(percent);
    lastPercent = percent;
  }
}, 15000);
4. Notifications

Use Tauri API:

import { sendNotification } from '@tauri-apps/api/notification';

function checkThresholds(percent) {
  if (percent >= 90) {
    sendNotification("Claude usage at 90% 🔴");
  } else if (percent >= 75) {
    sendNotification("Claude usage at 75% 🟡");
  }
}

5. System Tray (core UX)
Rust setup
use tauri::{SystemTray, SystemTrayMenu, CustomMenuItem};

let quit = CustomMenuItem::new("quit".to_string(), "Quit");

let tray_menu = SystemTrayMenu::new()
    .add_item(quit);

let tray = SystemTray::new().with_menu(tray_menu);
Handle events
.on_system_tray_event(|app, event| {
    match event {
        tauri::SystemTrayEvent::LeftClick { .. } => {
            let window = app.get_window("main").unwrap();
            window.show().unwrap();
            window.set_focus().unwrap();
        }
        tauri::SystemTrayEvent::MenuItemClick { id, .. } => {
            if id == "quit" {
                std::process::exit(0);
            }
        }
        _ => {}
    }
})
6. Hide Instead of Close
.on_window_event(|event| {
    if let tauri::WindowEvent::CloseRequested { api, .. } = event.event() {
        event.window().hide().unwrap();
        api.prevent_close();
    }
})

⚠️ Edge Cases (don’t skip these)
1. Not logged in

Detect:

if (document.body.innerText.includes("Sign in")) {
  return "NOT_LOGGED_IN";
}

→ Show notification or reopen window

2. Page stops updating

Fix:

setInterval(() => {
  webview.eval(`window.location.reload()`);
}, 5 * 60 * 1000);
3. Parsing fails
Return null
Don’t crash
Keep last known value
